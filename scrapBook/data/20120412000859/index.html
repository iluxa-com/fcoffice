Norton DNS 提供更快更安全的DNS

上一篇：互联网首场微播战 谁是杀毒大战的幕后推手 下一篇：360卫士拦截搜狗拼音 称其有木马风险

ashine发表于2010-05-30 20:00来源：www.safebeta.cn

Norton DNS是类似OpenDNS和Google DNS的域名解析服务，目前处于测试(beta)阶段。Norton DNS 宣称将提供更快，更安全，更可靠的DNS服务。
只需将DNS改为198.153.192.1和198.153.194.1 ，即可体验Norton DNS。

这里简单的对比一下OpenDNS,GoogleDNS,NortonDNS的速度吧。


OpenDNS:208.67.222.222,208.67.220.220
GoogleDNS:8.8.8.8,8.8.4.4
NortonDNS:198.153.192.1,198.153.194.1

使用命令"ping -n 20 IP" 测试20次Ping的时间。

OpenDNS:最短 = 218ms，最长 = 302ms，平均 = 251ms
GoogleDNS:最短 = 112ms，最长 = 216ms，平均 = 143ms
NortonDNS:最短 = 109ms，最长 = 200ms，平均 = 139ms

结果显示NortonDNS比GoogleDNS快那么一点点，当然这个结果也许不准确，大家可以试一试。

NortonDNS也提供了一个和Google Status类似的页面，用于报告DNS状况，地址是:http://status.nortondns.com/

NortonDNS的官方网站为：http://nortondns.com/
